# Quizline Backend

## Endpoints

- Auth
  - `POST /auth/signup`  
  - `POST /auth/login`
  - `POST /auth/token`

- Quiz 
  - `POST /quiz`: User's quizzes
  - `GET /quiz/new`: New quiz
  - `POST /quiz/edit/:id`: Edit quiz
  - `DELETE /quiz/`
